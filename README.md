## 💰 분야별 예산 체계 (NEW!)

### 예산 배정 원리
```
배정 예산 = MIN(총 수업 횟수 × 분야별 회당 지원금, 분야별 최대 상한)
```

### 분야별 회당 지원금 설정
| 분야 | 회당 지원금 | 최대 상한 | 특징 |
|------|------------|-----------|------|
| 한국요리문화 | 35,000원 | 800,000원 | 식재료, 조리도구 등 소모품 다량 소요 |
| 전통음악 | 30,000원 | 750,000원 | 전통악기 등 고가 교구 필요 |
| 전통문화예술 | 25,000원 | 600,000원 | 다양한 전통 문화 체험 도구 |
| 한국미술 | 22,000원 | 550,000원 | 미술 재료 및 도구 |
| 한국현대문화 | 18,000원 | 450,000원 | 현대 문화 체험 자료 |
| 한국어교육 | 15,000원 | 400,000원 | 교육 자료 및 교구 |
| K-Pop 문화 | 10,000원 | 300,000원 | 소모품이 적은 문화 체험 |

### 예산 배정 예시
- **K-Pop 분야, 30회 수업**: 30회 × 10,000원 = 300,000원 (상한 내)
- **한국요리문화, 30회 수업**: 30회 × 35,000원 = 1,050,000원 → 800,000원 (상한 적용)
- **한국어교육, 24회 수업**: 24회 × 15,000원 = 360,000원 (상한 내)

## 🛍️ 구매 방식 안내 (NEW!)

### 온라인 구매 📦
- **장점**: 배송 편의성, 관리자가 직접 처리
- **절차**: 신청 → 관리자 승인 → 관리자 구매 → 배송
- **적합한 경우**: 일반적인 교구, 온라인에서 구매 가능한 상품

### 오프라인 구매 🏪  
- **장점**: 현지에서만 구할 수 있는 특별한 교구 구매 가능
- **절차**: 신청 → 관리자 승인 → 학생 직접 구매 → 영수증 제출 → 비용 정산
- **적합한 경우**: 
  - 현지 전통 문화 용품
  - 현지에서만 구할 수 있는 재료
  - 신선 식품 등 배송이 어려운 상품
  - 현지 서점의 특별한 도서

### 영수증 제출 과정 📄
1. **관리자 승인** 후 '영수증 등록' 버튼 활성화
2. **영수증 촬영** 또는 이미지 파일 준비
3. **영수증 등록**:
   - 이미지 업로드 (드래그 앤 드롭 지원)
   - 구매일시 입력
   - 구매처 정보 입력
   - 추가 설명 (선택사항)
4. **제출 완료** 시 자동으로 구매완료 처리
5. **관리자 확인** 및 비용 정산 진행

## 📝 수업계획 기반 예산 배정 시스템

### 전체 프로세스
1. **학생**: 수업계획 작성 및 제출
2. **관리자**: 수업계획 검토 및 승인/반려
3. **시스템**: 승인 시 자동 예산 배정 (분야 × 수업 횟수)
4. **학생**: 배정된 예산 내에서 교구 신청 가능
5. **관리자**: 예산 사용 현황 모니터링

### 관리자 예산 설정 방법
1. **예산 설정 버튼** 클릭 (또는 `Ctrl + B`)
2. **분야별 설정**:
   - 회당 지원금: 해당 분야 수업 1회당 지원할 금액
   - 최대 상한: 개인당 최대 배정 가능한 예산
3. **설정 저장** 시 기존 승인된 학생들의 예산 자동 재계산

### 수업계획 승인 관리
1. **수업계획 관리 버튼** 클릭 (또는 `Ctrl + L`)
2. **제출된 수업계획 검토**:
   - 수업 횟수, 내용, 목표 확인
   - 예상 배정 예산 미리보기
3. **승인/반려 처리**:
   - 승인: 자동 예산 배정 및 교구 신청 허용
   - 반려: 상세한 반려 사유 입력

### 학생 사용법
1. **수업계획 작성**: 총 수업 횟수와 상세 계획 작성
2. **수업계획 제출**: 완료 후 관리자 승인 대기
3. **예산 배정 확인**: 승인 시 자동 배정된 예산 확인
4. **교구 신청**: 배정된 예산 내에서 교구 신청
5. **예산 모니터링**: 실시간 예산 사용 현황 확인

### 💰 새로운 예산 시스템 테스트

#### 단계 1: 예산 설정 관리 테스트
1. **예산 설정 모달 열기**:
   - 버튼 클릭: "예산 설정" 버튼
   - 단축키: `Ctrl + B`
2. **분야별 설정 변경**:
   - K-Pop 문화: 회당 지원금 15,000원, 최대 상한 350,000원으로 변경
   - 한국요리문화: 회당 지원금 40,000원으로 변경
3. **설정 저장 및 확인**:
   - "설정 저장" 버튼 클릭
   - "예산 설정이 저장되었습니다" 알림 확인
   - 기존 학생 예산 재계산 알림 확인

#### 단계 2: 수업계획 승인 관리 테스트
1. **수업계획 관리 모달 열기**:
   - 버튼 클릭: "수업계획 관리" 버튼
   - 단축키: `Ctrl + L`
2. **승인 대기 목록 확인**:
   - 대기 중/승인됨/반려됨 통계 확인
   - 김민수 학생의 수업계획 상태 확인
3. **수업계획 승인 처리**:
   - 김민수 수업계획의 "승인" 버튼 클릭
   - 확인 대화상자에서 "확인" 클릭
   - 예산 배정 완료 알림 확인

#### 단계 3: 학생 예산 현황 확인
1. **학생 로그인**: 김민수 (1998-03-15)
2. **예산 현황 확인**:
   - 대시보드 상단의 예산 정보 표시 확인
   - 분야명 (한국어교육) 표시 확인
   - 배정 예산: 24회 × 15,000원 = 360,000원 확인
   - 사용/잔여 예산 표시 확인
   - 진행률 바 색상 확인 (안전 = 녹색)

### 🎯 수업계획 → 예산 → 교구신청 전체 플로우 테스트

#### 시나리오: 새로운 학생의 전체 과정
1. **관리자로 반려 처리 테스트**:
   - 관리자 로그인 → 수업계획 관리
   - 김민수 수업계획 "반려" 버튼 클릭
   - 반려 사유: "수업 내용이 구체적이지 않습니다. 각 회차별 상세 계획을 작성해주세요."
   - 반려 처리 완료

2. **학생 반려 확인 및 재작성**:
   - 김민수 로그인 → 수업계획 상태 확인
   - "수업계획 반려됨" 상태 및 반려 사유 확인
   - 교구 신청 버튼 비활성화 확인
   - 수업계획 페이지에서 내용 수정 후 재제출

3. **관리자 재승인**:
   - 관리자 로그인 → 수업계획 관리
   - 수정된 수업계획 승인 처리
   - 예산 자동 배정 확인

4. **학생 교구 신청**:
   - 김민수 로그인 → 예산 현황 확인
   - "새 교구 신청" 버튼 활성화 확인
   - 교구 신청 (예: 한국어 교재 세트, 50,000원)
   - 예산 부족 시나리오: 400,000원 신청 → 예산 부족 경고 확인

### 📱 오프라인 구매 시스템 테스트

#### 단계 1: 오프라인 구매 신청
1. **학생 로그인**: 김민수
2. **교구 신청**:
   - "새 교구 신청" 클릭
   - 구매 방식: "오프라인 구매" 선택
   - 교구명: "베트남 현지 한국 문화 서적"
   - 가격: 30,000원
   - 참고 링크: 현지 서점 정보
   - 신청 완료

#### 단계 2: 관리자 승인
1. **관리자 로그인**
2. **신청 목록에서 확인**:
   - "오프라인 구매" 뱃지 확인
   - "승인" 버튼 클릭하여 승인

#### 단계 3: 영수증 등록
1. **학생 로그인**: 김민수
2. **영수증 등록**:
   - 승인된 항목에서 "영수증 등록" 버튼 확인
   - 버튼 클릭하여 영수증 등록 모달 열기
   - 이미지 파일 선택 또는 드래그 앤 드롭
   - 구매일시, 구매처 정보 입력
   - 영수증 제출 완료

#### 단계 4: 관리자 영수증 확인
1. **관리자 로그인**
2. **영수증 확인**:
   - 신청 목록에서 "영수증 보기" 버튼 클릭
   - 영수증 이미지 및 구매 정보 확인
   - "이미지 다운로드" 기능 테스트

### ⚙️ 고급 기능 테스트

#### 예산 시스템 스트레스 테스트
1. **여러 학생 동시 테스트**:
   - 다른 브라우저 탭에서 다른 학생으로 로그인
   - 동시에 교구 신청하여 예산 계산 정확성 확인

2. **예산 설정 변경 영향 테스트**:
   - 관리자: 분야별 지원금 대폭 변경
   - 기존 승인된 학생들의 예산 재계산 확인
   - 새로운 승인 시 변경된 설정 적용 확인

#### 키보드 단축키 테스트
1. **관리자 단축키**:
   - `Ctrl + B`: 예산 설정 모달
   - `Ctrl + L`: 수업계획 관리 모달
   - `Ctrl + T`: 테스트 모드 토글
   - `Ctrl + F`: 검색 포커스
   - `Ctrl + E`: Excel 내보내기

2. **학생 단축키**:
   - `Ctrl + N`: 새 교구 신청 (예산 승인 시에만)
   - `F5`: 데이터 새로고침

#### 데이터 내보내기 테스트
1. **Excel 내보내기**: `Ctrl + E`
2. **새로운 컬럼 확인**:
   - 배정예산, 사용예산, 잔여예산
   - 구매방식, 영수증번호, 영수증정보

### 🚨 오류 상황 테스트

#### 예산 관련 오류 처리
1. **예산 초과 신청**: 잔여 예산보다 큰 금액 신청 시 차단 확인
2. **수업계획 미승인 상태**: 교구 신청 버튼 비활성화 확인
3. **영수증 용량 초과**: 5MB 이상 파일 업로드 시 경고 확인

#### 네트워크 오류 시뮬레이션
1. **개발자 도구 → Network → Offline 설정**
2. **오프라인 상태에서 기능 동작 확인**
3. **에러 메시지 표시 확인**

### 🔧 문제 해결

#### 테스트 모드가 작동하지 않는 경우
1. 브라우저 새로고침 (`F5`)
2. 관리자 재로그인
3. 개발자 도구 콘솔에서 오류 메시지 확인

#### 예산 정보가 표시되지 않는 경우
1. 수업계획이 완료되고 승인되었는지 확인
2. 관리자에서 예산 설정이 올바른지 확인
3. 브라우저 캐시 삭제 후 재시도
