# 📝 업데이트 로그

> 프로젝트의 모든 변경사항을 기록하는 디렉토리

## 📅 업데이트 기록 규칙

### 파일명 형식
- `fix-YYYYMMDD-[문제설명].md` - 버그 수정
- `enhancement-YYYYMMDD-[개선내용].md` - 기능 개선
- `feature-YYYYMMDD-[기능명].md` - 새 기능 추가
- `core-update-YYYYMMDD-[변경내용].md` - 핵심 시스템 수정

### 업데이트 로그 템플릿
```markdown

[업데이트의 목적과 배경]
```
