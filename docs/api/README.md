# 📚 API 문서

> 프로젝트에서 사용하는 API 및 함수들의 문서화

## 🔧 핵심 API 모듈

### SupabaseAPI (`src/core/core-supabase.js`)
- 데이터베이스 연동 및 인증 처리
- CRUD 작업 및 실시간 데이터 동기화

### AuthManager (`src/core/core-auth.js`)
- 사용자 인증 및 세션 관리
- 권한 확인 및 접근 제어

### Utils (`src/core/util-common.js`)
- 공통 유틸리티 함수들
- DOM 조작, 데이터 포맷팅, 검증 등
